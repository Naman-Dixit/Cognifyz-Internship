## Track 1 — Software Development (Python)

### 1. Expense Tracker Console App — `ConsoleApp.py`
A CLI-based expense manager built with OOP.
- Full CRUD — Add, View, Update, Delete expenses
- Expense class with category, amount, and date fields
- Running total calculator

### 2. Enhanced Task Manager — `EnhanceCRUD.py`
A persistent task management system with JSON file storage.
- Full CRUD with auto-incremented Task IDs
- Priority levels (High / Medium / Low) and due dates
- Search by keyword and filter by status (Pending / Completed)
- JSON persistence across sessions

### 3. Temperature Converter — `Temperature.py`
Simple CLI tool for bidirectional temperature conversion.
- Celsius ↔ Fahrenheit

### 4. Number Pattern Generator — `NumberPattern.py`
Prints a symmetric diamond-shaped number pattern.
- User-defined size input
- Expanding and contracting rows (upper + lower half)

### 5. Text Adventure Game — `Game.py`
A branching story-based CLI game set in a haunted mansion.
- Multiple rooms with unique endings (Library, Kitchen, Basement)
- Code puzzle in the basement — crack `777` to escape

### 6. Web Scraper — `Webscrapper/`
A Streamlit-powered web scraping app using Selenium.
- URL input via Streamlit UI
- ChromeDriver-based page source extraction
- Stack: `selenium`, `streamlit`, `beautifulsoup4`, `langchain`

**Setup:**
```bash
cd Webscrapper
pip install -r requirements.txt
streamlit run Main.py
```
